# Stadhuisbericht

Maandelijkse nieuwsbrief van Lokale Lobby over wat er in het Nijmeegse 
stadhuis gebeurt en wat dat voor MKB-ondernemers betekent.

Geschreven door Maarten Bakker, oud-raadslid van de gemeente Nijmegen 
met tien jaar recente lokale politieke ervaring.

## Structuur

Elke editie heeft een eigen map onder `edities/` met als naamconventie 
`JJJJ-MM`. In die map staat de HTML-versie van de nieuwsbrief zoals 
verzonden via MailerLite.

## Edities

- `edities/2026-05/` editie mei 2026
